A simple, no-nonsense WordPress gallery plugin featuring a built-in lightbox and grid layout.

== Description ==

- Create galleries as a custom post type (KTS Gallery).
- Select & upload images via the Media Library, drag to reorder.
- Basic layout settings (columns, gap, image height).
- Copy the shortcode from the editor sidebar: [kts_gallery id="123"]
- Duplicate galleries with one click from the list table.
- List table shows preview thumbnail and shortcode.

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload Plugin and activate.
2. Go to **KTS Gallery → Add New** to create your first gallery.
3. Insert the gallery with the shortcode `[kts_gallery id="YOUR_ID"]`.

== Frequently Asked Questions ==

= How do I change columns/gap/height per gallery? =
Edit the gallery and use the **Gallery Settings** meta box. You can also override in the shortcode:
[kts_gallery id="123" columns="4" gap="12px" height="220px"]

= Does it support lightbox? =
Yes, the lightbox is built-in and enabled by default. You can turn it off in settings if needed.
